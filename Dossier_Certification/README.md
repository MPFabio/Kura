## Contexte du projet

Le projet Kura est une plateforme DevOps unifiée permettant la gestion centralisée de :
- Clusters Kubernetes
- États Terraform
- Jobs Ansible
- Pipelines CI/CD
- Métriques et alertes

Cette plateforme répond au besoin d'avoir une vue centralisée sur plusieurs outils DevOps dispersés, réduisant la charge cognitive des équipes et facilitant les diagnostics cross-systèmes.
